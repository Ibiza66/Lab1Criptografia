# Laboratorio 1 - Criptografía y Seguridad en Redes

## Descripción
Este repositorio contiene el desarrollo del Laboratorio 1, incluyendo:
- Informe en PDF
- Código en Python
- Imágenes de evidencias

- `codigo/`: scripts de cifrado, envío ICMP y descifrado
- `imagenes/`: capturas utilizadas en el informe
